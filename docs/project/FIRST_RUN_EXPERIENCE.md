# First Run Experience

Lodestone's first-run experience is the foundation of a strong account journey. It must be clear, guided, and aligned with the platform's strategy-first philosophy.

## Objectives

- Establish the player's identity and account context.
- Capture the chosen game and goal.
- Capture the desired optimization level.
- Capture data provider preferences.
- Seed the Account Engine with initial progress data.

## User Flow

1. Welcome and mission statement.
2. Choose game:
   - RuneScape 3
   - Old School RuneScape
3. Choose goal:
   - Quest Cape
   - Master Quest Cape
   - Completionist Cape
   - Trimmed Completionist Cape
   - Custom Goal
4. Choose optimization level:
   - Explorer (~90%)
   - Pathfinder (~95%)
   - Navigator (~100%)
5. Choose data providers:
   - Manual Entry
   - RuneMetrics
   - Wiki Sync
   - CSV Import
   - Smart Capture (OCR)
6. Create account:
   - account name/email
   - password or authentication flow
7. Confirm initial import and begin planning.

## Experience Principles

- The first-run flow should be concise and explain the value of each choice.
- The system must never assume the user has perfect input data.
- Users should be able to start with manual entry and improve their profile later.
- Provider selection should be optional when no data source is immediately available.
- The experience should make it clear that Lodestone is a recommendation engine, not a catalog.

## Initial Data Capture

- Game selection must determine the supported activity and provider sets.
- Goal selection should determine planning templates and journey targets.
- Optimization level should influence Navigator scoring and recommendation strictness.
- Provider selection should define how account progress is populated.
- Account creation should preserve user progress and provider choices.

## Success Criteria

- The user has a valid account profile.
- The user has selected a goal and optimization mode.
- The user has chosen at least one data provider or opted for manual entry.
- The system can generate an initial recommendation or action plan.
- The user understands that the platform will compute strategy from facts.
