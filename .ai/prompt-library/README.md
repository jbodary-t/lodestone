# Prompt Library

This directory is reserved for reusable prompt templates and guidelines for future AI assistants.

## Purpose

- Store prompt patterns for common tasks.
- Capture conventions for architecture reviews, bug fixes, and developer experience improvements.
- Help future AI and human contributors follow Lodestone's engineering rules.

## Usage

Add prompt templates here only when they are stable and broadly useful.
Keep templates focused on maintainability, correctness, and architecture.
